# Проект: Место
Проектная работа курса "Яндекс Практикум"
Сайт с галерей фотографий, где можно будет редактировать информацию об авторе, добавлять фото и ставить лайки.

### Технологии

* HTML
* CSS
* JS
* Grid, Flexbox, Adaptive, Buttons
